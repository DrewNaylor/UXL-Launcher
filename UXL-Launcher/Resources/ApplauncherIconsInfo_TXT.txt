The applauncher-icons project provides most of the icons used in UXL Launcher, with the exception of some icons. These include, but are not limited to, the UXL Launcher logo and the About window banner.
Link to applauncher-icons project repository: https://github.com/DrewNaylor/applauncher-icons
Link to applauncher-icons version 2.0 GitHub Releases page: https://github.com/DrewNaylor/applauncher-icons/releases/tag/v2.0

The version of applauncher-icons used in UXL Launcher has not been modified from the original source.
